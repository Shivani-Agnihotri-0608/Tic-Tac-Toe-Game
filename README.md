# Tic-Tac-Toe Game

This is a classic Tic-Tac-Toe game built using **HTML, CSS, and JavaScript.  
Players can take turns clicking on the grid to place **X** or **O**. The game detects a **win, loss, or draw** automatically and allows players to restart the game.
